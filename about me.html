About Me

My name is Qanita, a student of class 10 PPLG 3 at SMK Negeri Tengaran, majoring in Software Engineering. Creating this portfolio website is one of my valuable experiences and a school assignment I completed to introduce myself and showcase the skills I have. At first, I found it quite difficult because this was my first time using HTML and CSS programming languages to structure and beautify the appearance of a web page. I chose a soft pink theme because I find this color looks elegant, beautiful, and comfortable to look at. During the creation process, I learned how to arrange the page structure, input my complete personal data, manage the layout, and add animations to make the display look more lively. I also encountered several obstacles, such as messy layouts and links that did not work properly, but I kept trying to find solutions and fix them until the results matched what I wanted. Through this experience, I not only understood how to create a web page, but I also learned to be more thorough, patient, and creative in completing a task. This work motivates me even more to keep deepening my knowledge of programming and design to achieve my dream of becoming a reliable software developer in the future.
